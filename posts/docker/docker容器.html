查看所有容器

查看运行的容器

命令：docker ps

作用：查看所有正在运行的容器

查看所有的容器

命令：docker ps -a

作用：查看所有的容器，包含正在运行的和没有运行的。

创建容器

新建容器

命令：docker [container] create 容器名

作用：新建一个容器，但处于停止状态，需要 docker start 命令来启动它。

启动容器

命令：docker [container] start

新建并启动容器

除了创建容器后通过start命令来启动，也可以直接新建并启动容器。

命令：docker [container] run

作用：新建并执行容器。等价于create和start命令。

停止容器

暂停容器

命令：docker [container] pause

作用：暂停运行中的容器

处于paused状态的容器，可以使用docker [container] unpause 命令恢复到运行状态。

终止容器

命令：docker [container] stop

作用：终止运行中的容器。

该命令会首先向容器发送SIGTERM信号，等待一段时间后（默认是10秒），再发送SIGKILL信号来终止容器。
处于终止状态的容器，可以通过docker [container] start 命令重新启动。

重启容器

命令：docker [cntainer] restart

作用：将一个运行状态的容器先终止，然后再重新启动。

删除容器

命令：docker [container] rm

作用：删除处于终止或退出状态的容器

主要选项：

  • -f ,–force=false：是否强制终止并删除一个运行中的容器；
  • -l , –link=false：删除容器的连接，但保留容器；
  • -v，–volumes=false：删除容器挂载的数据卷。
